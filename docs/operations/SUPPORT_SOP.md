# Support Standard Operating Procedures (SOP)

This document defines how MentalFlow handles user inquiries, technical bugs, and clinical emergencies.

## 🏢 Support Tiers

### Tier 1: General Support (Agent)
- *Scope*: Password resets, billing questions, app navigation.
- *Goal*: Response < 4h. Resolution < 24h.

### Tier 2: Technical Support (Engineer)
- *Scope*: Bugs, sync errors, RL model inconsistencies.
- *Goal*: Response < 12h. Resolution < 3 business days.

### Tier 3: Clinical Support (Therapist)
- *Scope*: Clarifying assessment scores, clinical guidance.
- *Goal*: Response < 24h.

---

## 🚨 Clinical Escalation Protocol (CRITICAL)

**Trigger**: If a user reports immediate self-harm in any channel (Email, Chat, Review) or triggers a PHQ-9 Item 9 "Crisis Detect".

### Immediate Actions:
1. **Emergency Resources**: Auto-response with 988 Lifeline and Crisis Text Line.
2. **Support Agent Audit**: Flag user account in the Therapist Dashboard immediately.
3. **Clinical Review**: Escalate to the On-Call Clinical Director for immediate outreach within 4 hours.
4. **Law Enforcement**: Only if immediate danger is confirmed and non-emergency methods fail (refer to HIPAA Privacy Rule Section 164.512(j)).

## 📊 Support SLAs & Metrics
- **CSAT Target**: > 4.6 / 5.0
- **FRT (First Response Time)**: < 2 hours during business window.
- **Repeat Ticket Rate**: < 10%.
