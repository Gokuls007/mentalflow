# Customer Onboarding Strategy: The MentalFlow Journey

This document outlines the critical first 30 days of the user journey to ensure clinical engagement and long-term retention.

## 🏁 Day 1: The Activation Loop
**Goal**: Move the user from "Registered" to "First Activity Complete" within 120 minutes.

- **T + 0m**: Welcome Email (Clinical trust building).
- **T + 5m**: Informed Consent & Profile Setup (In-app).
- **T + 15m**: Baseline Assessments (PHQ-9/GAD-7).
- **T + 25m**: Personalized Value Clarification (Values exercise).
- **T + 30m**: **First Activity Assignment** (Low-barrier, e.g., "5-minute mindful walk").
- **T + 60m**: Celebration of first 25 XP earned.

## 📈 Days 2–7: Momentum Building
- **Push Notification Cadence**: Afternoon nudges for mood check-ins.
- **Micro-Achievements**: Unlock "First Streak" badge on Day 3.
- **Engagement Trigger**: If 0 activities completed by Day 4, send recovery email with a simplified activity suggestion.

## 🛡️ Days 8–30: Clinical Maturation
- **Week 2**: Introduce the "Clinical Insights" leaderboard highlighting progress trends.
- **Day 21**: Habit Formation Check-in (Self-reported engagement evaluation).
- **Day 30**: Comprehensive Mood Progress Report (Shareable with primary care physician).

## 📊 Key KPIs
| Metric | Threshold |
| :--- | :--- |
| **Registration -> Consent** | > 85% |
| **Consent -> Assessment** | > 75% |
| **Day 1 Activity Completion**| > 50% |
| **Day 7 Retention (W1)** | > 40% |
| **NPS (Day 30)** | > 45 |
